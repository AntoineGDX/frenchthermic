Energy-class chip (Diagnostic de Performance Énergétique), the arrow shape French homeowners recognise from the official label.

```jsx
<DpeBadge grade="B" label="Après travaux" />
<DpeBadge grade="F" size="lg" />
```

`grade` A–G drives the color (green→red). D/E switch to dark text for contrast. Use it on audit results, listings and before/after comparisons.
